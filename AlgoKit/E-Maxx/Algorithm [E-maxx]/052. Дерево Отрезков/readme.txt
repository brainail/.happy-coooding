Дерево отрезков позволяет делать операции :
 0. Создать дерево отрезков ( Time: O(~NlogN); Memory: O(~N*(~3..4)) )
 1. Увеличить промежуток чисел от L до R на число W. ( O(~logN) )
 2. Найти минимум, максимум, сумму - на отрезке от L, до R. ( O(~logN) )
 3. Кол-во минимальных, максимальных элементов на промежутке от L до R. ( O(~logN) )
